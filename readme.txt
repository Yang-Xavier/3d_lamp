Running followed code to start program.

javac Anilamp.java 

java Anilamp


In the basisObj package, some basis models such as cube, cylinder, sphere would be in this package
In the gmaths package, that is the extra library for matrix and vector calculation.
In the lamp package, all components of the lamp are placed in this package.
In the scene package, all static model would be placed in this package.
In the tool package, there are some common class and super-class which are generally used to build and render the model.
